# 📊 SQL Sales Dashboard Project

## 📌 Overview
This project demonstrates a complete SQL-based sales dashboard using:
- joins
- aggregations
- group by
- filters
- having
- window functions

## 🧱 Database Schema
- customers
- products
- orders
- order_items

## 📊 Key Insights
- total sales calculation
- category-wise performance
- city-wise sales
- monthly trends
- top selling products
- high-value customers

## ⚙️ How to Run
1. run 01_create_tables.sql
2. run 02_insert_data.sql
3. run 03_dashboard_queries.sql
4. run 04_advanced_queries.sql
